# TO RUN THE PROGRAM
--> First ensure the following libraries are installed: 
   `streamlit`, `joblib`, `tensorflow`, `numpy`, `pandas`, `sklearn`, `seaborn`, and `matplotlib`
   
1. Make a new directory in your VSCode folder and clone the repository
2. Run the "setup_models.py" file to train the models and initialize the scaler
3. Open a terminal window within the "StarCraftPredictor" directory and type
   `python -m streamlit run starcraft_rank_app.py`
4. Follow the prompts from streamlit, then the webapp will automatically open
5. Input known data or your own stats to view what rank you are

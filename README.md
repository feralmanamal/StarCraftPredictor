# TO RUN THE PROGRAM
--> First ensure the following libraries are installed: 
   `streamlit`, `joblib`, `tensorflow`, `numpy`, `pandas`, `sklearn`, `seaborn`, and `matplotlib`
   
1. Make a new folder in your VSCode directory (your main folder) and clone the repository inside it (your repository folder)
2. Move the "starcraft.csv" file from the repository folder into the main folder
3. Run the "setup_models.py" file to train the models and initialize the scaler
4. Open a terminal window within the main folder and type
   `python -m streamlit run starcraft_rank_app.py`
5. Follow the prompts from streamlit, then the webapp will automatically open in your browser

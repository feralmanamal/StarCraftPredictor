# TO RUN THE PROGRAM
--> First ensure the following libraries are installed: 
   `streamlit`, `joblib`, `tensorflow`, `numpy`, `pandas`, `sklearn`, `seaborn`, and `matplotlib`
   
1. Clone the repository within your IDE's main directory
2. Run the "setup_models.py" file to train the models and initialize the scaler
3. Open a terminal window within the main folder and type
   `python -m streamlit run starcraft_rank_app.py`
4. Follow the prompts from streamlit, then the webapp will automatically open in your browser
